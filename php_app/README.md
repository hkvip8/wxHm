PHP 版本微信活码系统（基础骨架）

说明：
- 这是一个最小可运行的 PHP 项目骨架，包含用户中心、后台管理、套餐和支付回调示例。
- 支付集成（易支付/支付宝）为示例代码，需要在 `src/config.php` 填写商户信息和回调地址，并根据支付通道文档适配签名验证。

快速开始：
1. 在服务器上准备 PHP >=8、pdo_mysql 扩展和 MySQL 数据库。
2. 导入数据库结构：
   mysql -u root -p < sql/schema.sql
3. 配置 `src/config.php` 数据库和支付参数。
4. 将 `php_app/public` 目录作为网站根目录（Nginx/Apache）。

文件说明：
- public/index.php: 前端路由，处理登录、注册、用户中心、后台入口和支付回调。
- src/: 应用逻辑（数据库、认证、支付、套餐等）。
- templates/: 简单的视图模板。
- sql/schema.sql: 数据库结构。
