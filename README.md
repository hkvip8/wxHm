# 微信群活码系统 (多文件增强版)

本工具旨在解决微信群二维码 7 天失效的问题。采用“多码轮转”逻辑，确保前端用户始终看到最新、有效的入口。

## ⚙️ 运行环境
- Python 3.x
- Flask

## 🚀 启动步骤
1. 安装依赖：`pip install Flask`
2. 运行程序：`python app.py`
3. 访问主页：`http://localhost:5000`
4. 管理后台：`http://localhost:5000/admin` (默认密码: `admin123`)

## 🛠️ 工作原理
1. **上传**：管理员通过后台上传图片，文件以 `qr_时间戳.png` 格式存入 `uploads/` 文件夹。
2. **检索**：用户访问首页时，程序自动扫描文件夹，按时间倒序排列，取出最新的一张。
3. **校验**：若最新图片超过 7 天，程序会自动尝试寻找“次新”的一张，并清理掉过期文件。
4. **防刷**：图片 URL 附带毫秒级随机参数，绕过微信内置浏览器的强缓存。

## 🛡️ 安全建议
正式上线前，请务必在 `app.py` 中修改 `ADMIN_PASSWORD`。
